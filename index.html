Work

Harmonize - Github Repo

Harmonize is a chord helper app that allows users to pick a starting note and chord quality, and from there suggests chords that would make sense to go to next in a progression. I wanted to create a tool for people that know some chords, but aren't really sure how to put them together in a practical and musical way.Users can save favorite progressions to their profile for future reference. Made using AngularJS, Express/NodeJS, MongoDB, HTML, CSS, JavaScript, Bootstrap and TeoriaJS.

Musicians Meet-Up - Github Repo

Musicians Meet-Up is an app where musicians of all levels can connect and collaborate. Inspired by aspects of Facebook and Craigslist, users are able sign-up and create a profile. They can create and respond to posts in the Classifieds section and also search through all the users (the community). Made using Express/NodeJS, Sequelize, HTML, CSS, JavaScript, jQuery, Bootstrap, Bcrypt, Cloudinary and MailgunJS.

Hangman - Github Repo

This was my first project during my time at General Assembly. I wanted to re-make the classic word guessing game and add a Western twist. My main focus of this project was to make something functional that looked good, but was also fun to play. Made using jQuery, Javascript, CSS, HTML, Lazy Line Painter and the Wordnik API.

Concrt - Github Repo

My first and only group project at GA. Concrt is a full stack web application that allows users to discover and share local concerts using the Jambase API. Concerts can be saved to a user calendar that can be referenced from the user profile. Users can also "follow" eachother and these relationships are displayed on their profile page. Made using Ruby, Rails, PostgreSQL, HTML, CSS, JavaScript, jQuery, Bootstrap, D3JS and the Jambase API.

Jeopardy React - Github Repo

A small project I made to familiarize myself with the basic concepts of ReactJS. I used the jService API to generate the random question data and also used Swipe to enable swiping back and forth between questions and answers (Please Note: Swiping only works on mobile devices). Made using ReactJS, Swipe, jService, Bootstrap, Javascript, jQuery, HTML and CSS
