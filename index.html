About

Whether it's playing music or writing code, I am always creating. A drummer turned web dev, I feel that my musical background gives me a unique perspective on the tech industry. I love collaboration and creating as a team. Technologies I have experience with include Javascript, HTML, CSS, Node, Express.js, Angular.js, Ruby, Rails, Postgres and SQL.

When I'm not coding you can find me playing music, enjoying the outdoors, or gaming.

Work

Hangman - Github Repo

This was my first project during my time in General Assembly. I wanted to re-make the classic word guessing game with a Western twist. My main focus of this project was to make something functional that looked good, but that was also fun to play. Made using jQuery, Javascript, CSS, HTML, Lazy Line Painter and the Wordnik API.

Musicians Meet-Up - Github Repo

Musicians Meet-Up was inspired by my desire to create a social site where musicians of all levels could connect and collaborate. Inspired by aspects of Facebook and Craigslist, users are able sign-up and create a profile. They can also create and respond to posts in the Classifieds section and search through all users (the community). Made using Express/Node.js, Sequelize, HTML, CSS, JavaScript, jQuery, Bootstrap, Cloudinary and Mailgun.js.

Concrt - Github Repo

My first and only group project at GA. Concrt is a full stack web application that allows users to discover and share local concerts using the Jambase API. Concerts can also be saved to a user calendar that can be referenced from the user profile. Made using Ruby on Rails, PostgreSQL, Ruby, HTML, CSS, JavaScript, jQuery, Bootstrap, D3.js and the Jambase API.

Harmonize - Github Repo

A chord helper app that allows users to pick a starting note and chord quality, and from there suggests chords that would make sense to go to next in a progression. Users can save favorite progressions to their profile for future reference. Made using Angular.js, Express/Node.js, MongoDB, HTML, CSS, JavaScript, Bootstrap, and Teoria.js.
