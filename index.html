Projects/For Fun

React Movie Search App - Github | Demo

I created a few adaptations of this app during my time at General Assembly Seattle and recently decided to create a React version using Babel, Webpack, and some new features of ES2017. It's a simple app that allows users to search the OMDB movie database using keywords. Each successful query returns up to 10 results. Users can then click on a result card to see more information on a particular film.

Tech Used: React, Babel, Webpack, Materialize, Node/Express, OMDB API

ES6 Pokemon App - Github | Demo

A simple pokemon app that utilizes new technologies including ES6, Flexbox and new CSS3 features. Search for info on a specific Pokemon or build an effective team against certain types of Pokemon. Made using the Pokemon API. Features used include: arrow functions, const/let, promises (asynchronous API calls), spread operator, fetch, css variables, and fetch.

Tech Used: HTML5, CSS3, JS(ES6), Handlebars, Node/Express

Linear Drum Beats - Github | Demo

A Linear Drumbeat is a drumbeat where no two sound sources strike at the same time. I made this simple VueJS project to create random linear patterns that I normally wouldn't think of. The app creates and draws the beat using drum notation. Users can also hit the play button to hear how the beat would sound if played by a drummer.

Tech Used: HTML5, SCSS, JS, Bootstrap4, VueJS, Gulp, Express/NodeJS

Harmonize - Github | Demo

A chord helper app that allows users to pick a starting note and chord quality, and from there suggests chords that make sense to go to next in a progression. I wanted to create a tool for people that know some chords, but aren't sure how to put them together in a practical and musical way. Users can save favorite progressions to their profile for future reference.

Tech Used: HTML5, CSS3, JS, Bootstrap3, TeoriaJS, AngularJS, Express/NodeJS, MongoDB/Mongoose

Bog Adventure - Github | Demo

A simple Ruby on Rails app with CRUD functionality. Users can create and edit bog creatures as well as add tags to each creature. Creatures and tags can also be deleted.

Tech Used: HTML5, CSS3, Ruby, Ruby on Rails, Postgres

Hangman - Github | Demo

My first project during my time at General Assembly. I wanted to re-make the classic word guessing game and add a Western twist. The focus of this project was to make something functional that looked good and was also fun to play.
